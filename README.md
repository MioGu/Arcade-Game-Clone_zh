# 前端纳米学位街机游戏克隆项目

* 这是一个用来学习**面向对象**思想的练习项目

## How To Play
双击index.html（游戏入口文件），打开游戏。 通过键盘 **上 下 左 右** 来操纵玩家躲避敌人。一旦被敌人碰撞到，玩家便会回到初始位置。

## 文件结构
```shell
├── js
│   ├── app.js       游戏对象封装与创建
│   ├── engine.js    游戏主循环引擎
│   └── resources.js    游戏素材加载工具
├── images    游戏素材
├── css       舞台样式
├── index.html   游戏入口文件
└── README.md
```
